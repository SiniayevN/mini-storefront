Mini-Storefront

A simple Next.js storefront demo that shows products, lets users filter them by category or price, and manage a cart with stock limits.

How to run locally

1. npm install
2. npm run dev
3. Open [http://localhost:3000](http://localhost:3000)

How to build
npm run build
npm start

Features

* Filter products by category and price
* Add and remove items from the cart
* Prevent adding more than available stock
* Responsive grid layout
* Clean, minimal Tailwind styling

Known issues
Cart resets on page reload
No backend database (mock API only)
